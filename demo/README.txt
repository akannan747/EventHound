Instructions to run EventHound. 

1. In command line, execute "python form.py".
2. Click OK in the first welcome window.
3. Enter your name and click OK to continue.
4. Enter your search query (1 word!) and click OK to continue.
5. Enter the number of results you would like to see and click OK to continue.
6. The algorithm will run.
7. Once it is done running, it will display the top X event links in a new window.
8. You can click on the event links and it will open in the browser.

The steps are also documented as screenshots within in this demo folder.
